# Douyin Hot Spot Analyzer - 设计文档

**日期**: 2025-01-15  
**状态**: 已批准

---

## 1. 项目概述

**项目名称**: Douyin Hot Spot Analyzer

**核心功能**:
1. 实时获取抖音热搜话题、视频详情、创作者信息
2. 生成结构化 Markdown 文档供人工阅读
3. 提供 Web 界面浏览和管理热点数据

**目标用户**: 内容创作者、视频策划人员

---

## 2. 技术栈

**后端**:
- Node.js + Express
- TypeScript
- SQLite（轻量级存储）
- 第三方抖音 SDK

**前端**:
- React + TypeScript
- Vite（构建工具）
- Tailwind CSS（样式）

**其他**:
- Docker（可选部署）

## 4. API 设计

### 热点相关

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/hot/topics` | GET | 获取热搜话题列表 |
| `/api/hot/videos` | GET | 获取热门视频列表 |
| `/api/hot/creators` | GET | 获取热门创作者 |
| `/api/hot/refresh` | POST | 手动刷新热点数据 |

### 文档相关

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/docs/today` | GET | 获取今日 Markdown 文档 |
| `/api/docs/export` | GET | 导出 Markdown 文件 |

### 响应格式

```json
{
  "success": true,
  "data": [...],
  "error": null
}
```

## 5. 前端页面结构

### 页面列表

1. **热点概览** (`/`) - 热搜话题 + 热门视频 + 热门创作者卡片
2. **话题详情** (`/topic/:id`) - 单个话题详细信息
3. **文档中心** (`/docs`) - 历史 Markdown 文档列表
4. **设置** (`/settings`) - 配置管理

### 核心组件

| 组件 | 描述 |
|------|------|
| `HotTopicCard` | 热搜话题卡片（排名、话题名、热度值） |
| `VideoCard` | 视频卡片（封面、标题、作者、互动数据） |
| `CreatorCard` | 创作者卡片（头像、昵称、粉丝数） |
| `MarkdownPreview` | Markdown 预览 + 复制/下载按钮 |
| `RefreshButton` | 手动刷新按钮 + 状态指示 |

## 6. 数据模型

### HotTopic - 热搜话题

```typescript
{
  id: string;
  word: string;          // 话题名称
  hotValue: number;      // 热度值
  rank: number;          // 排名
  link: string;          // 话题链接
  fetchedAt: Date;       // 获取时间
}
```

### Video - 热门视频

```typescript
{
  id: string;
  title: string;         // 视频标题
  author: Creator;       // 作者信息
  likes: number;         // 点赞数
  comments: number;      // 评论数
  shares: number;        // 分享数
  coverUrl: string;      // 封面图
  fetchedAt: Date;
}
```

### Creator - 创作者

```typescript
{
  id: string;
  nickname: string;      // 昵称
  avatar: string;        // 头像
  followers: number;     // 粉丝数
  verified: boolean;     // 是否认证
  fetchedAt: Date;
}
```

## 8. 错误处理

| 场景 | 处理方式 |
|------|---------|
| 抖音 API 请求失败 | 返回缓存数据（如有）+ 错误提示 |
| 网络超时 | 重试 3 次，失败后返回友好错误 |
| 数据解析失败 | 记录日志，返回部分数据 |
| 前端 API 调用失败 | 显示 toast 提示 + 重试按钮 |
